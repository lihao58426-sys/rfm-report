# RFM 客户价值分析报告

纯手写 Vanilla JS + CSS 的 RFM（Recency / Frequency / Monetary）客户价值分析报告。支持 CSV 导入、8 类客户分群、行动优先级矩阵、ROI 估算。

## 功能

- **RFM 分析模型**：最近消费时间（R）、消费频次（F）、消费金额（M）
- **8 类客户分群**：高价值/重点发展/重点保持/潜在客户/新客户/流失预警/一般维护/挽留客户
- **可视化**：月度淡旺季堆叠柱状图、行动优先级矩阵（P0-P3）
- **运营策略**：每类客户给出可落地运营建议 + ROI 估算
- **数据驱动架构**：修改 CONFIG 对象即可复用到其他场景
- **响应式设计**：手机 / PC 自适应

## 技术栈

Vanilla JS · CSS · HTML5 · RFM 分析模型

## 使用

1. 用浏览器打开 `脱敏数据报告.html`
2. 在 CONFIG 对象中填入 CSV 数据
3. 报告自动计算并展示所有分析结果
